# Лабораторная 1 по курсу ООП

## Блок 1: Введение в C#

Наша цель в этом блоке — освоить/вспомнить базовые синтаксические элементы языка программирования C#. А также познакомиться с написанием тестов с использованием фреймворка NUnit.

### Темы заданий

1. Простейший ввод-вывод.
2. Функции и их параметры.
3. Условные операторы.
4. Циклы.
5. Параметры командной строки, длинные целые.

### Начисление баллов за занятие

* (C1 в ведомости) = Task1 (1 балл) + Task2 (1 балл) + Task3 (2 балла)
* (C2 в ведомости) = Task4 (2 балла) + Task5 (2 балла)

## Блок 2: Обработка строк

Наша цель в этом блоке — освоить элементы стандартной библиотеки языка C#, связанные с обработкой строк
   
### Темы заданий

1. Простейшие приемы обработки строк (String).
2. Формирование длинных строк (StringBuilder).

### Начисление баллов за занятие

* (C3 в ведомости) = Task1 (2 балла) + Task2 (2 балла)
